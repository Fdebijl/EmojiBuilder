Save the emoji as an SVG-file that you can edit anytime by coming back here and selecting 'File > Open'.

Note: this isn't real SVG, so please dont try to open it in your browser. Unless it's only one emoji, in which case it will work fine.

EmojiBuilder by @fdebijl

For more info, or to help me out finish this site, visit the GitHub page. Instructions: Click any emoji on the right or drag them onto the drawboard to start editing. You can select any path by clicking on it once. To deselect a path, click anywhere else on the drawboard. Or on a different path to select that one. To delete a path, press 'Delete' on your keyboard or select 'Edit > Delete path' To save your creation as an image file (.png) to share with other, select 'File > Save As... > PNG'. If you want to save an emoji for later editing, you may select SVG in the same dialog. In order to open the saved SVG file, select 'File > Open...' and open your SVG file. To delete the current emoji, select 'File > New'. To hide this pop-up and start editing, click the button below.

Make Emojibuilder available offline

Emojibuilder will always attempt to cache all emoji and files required to function, so the site will load faster on your next visit and will even work offline. Your browser may however clear this cache at any time, you can prevent this by granting persistent storage permission. This will maintain the cache (current size: empty) for your next visit.
